# NUMBER SYSTEM CONVERTER USING JAVA

## Overview

The **Number System Converter** is a simple program that converts numbers between four common number systems used in computing:

* **Binary (Base 2)**
* **Octal (Base 8)**
* **Decimal (Base 10)**
* **Hexadecimal (Base 16)**

## Features

* Convert **Binary to Decimal, Octal, and Hexadecimal**
* Convert **Decimal to Binary, Octal, and Hexadecimal**
* Convert **Octal to Binary, Decimal, and Hexadecimal**
* Convert **Hexadecimal to Binary, Octal, and Decimal**

---

## Number Systems Explained

### Binary (Base 2)

Binary uses only **two digits: 0 and 1**.
It is the primary number system used by computers.

---

### Octal (Base 8)

Octal uses **eight digits: 0–7**.
It is sometimes used as a shorter representation of binary numbers.

---

### Decimal (Base 10)

Decimal uses **ten digits: 0–9**.
This is the standard number system used in everyday life.

---

### Hexadecimal (Base 16)

Hexadecimal uses **sixteen symbols: 0–9 and A–F**.

```
A = 10
B = 11
C = 12
D = 13
E = 14
F = 15
```

## Example Conversions

| Input         | Binary | Octal | Decimal | Hexadecimal |
| ------------- | ------ | ----- | ------- | ----------- |
| 10 (Decimal)  | 1010   | 12    | 10      | A           |
| 1111 (Binary) | 1111   | 17    | 15      | F           |
| 77 (Octal)    | 111111 | 77    | 63      | 3F          |
